📝 Kobo2Docx

Automatically convert KoboToolbox & ODK XLSForm questionnaires into clean, publication-ready Word (.docx) documents.

Download Zip View on GitHub

Key Features

🌐 Multi-Language Support

Auto-detects label::Language headers and builds individual Word documents for each language seamlessly.

🎨 Clean Layouts

Creates full-width shaded section header rows for groups and formatted tables with S.N., Questions, and Response Options.

🔀 Smart Logic Formatting

Questions dependent on skip logic (relevant) are automatically indented and styled in fainter 10pt text.

⚡ One-Click Local Runner

Includes a run.bat file for Windows users—double-click to convert questionnaires locally without touching terminal code.

How to Use Locally

  1. Download or clone the repository to your computer.
  2. Place your Kobo Excel forms (.xlsx) inside the inputs/ folder.
  3. Double-click run.bat (or run python main.py in terminal).
  4. Collect your formatted Word documents in the outputs/ folder.
